Highlight errors on Jenkinsfile.

Execute jenkins linter to detect all errors on Jenkinsfile.

See https://github.com/jenkinsci/jenkinsfile-runner for more information.

Example: 


pipeline {
	agent hola
	triggers {
		cron('H */4 * * 1-5')
	}
	stages {
		stage('Example') {
			steps {
				echo 'Hello World'
			}
		}
	}
}
